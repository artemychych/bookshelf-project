# BookShelf

Приложение для учёта прочитанных книг с возможностью ставить рейтинги, писать рецензии и добавлять цитаты.

## Особенности
- Каталог книг с обложками из Google Books API
- Фильтры по жанрам, статусам (прочитано/читаю/хочу прочитать)
- Рецензии и рейтинги
- Статистика (книг в год, по жанрам)
- Экспорт данных в CSV/JSON

## Технологии
- Frontend: React
- Backend: Node.js + Express
- Database: PostgreSQL + Sequelize
- CI/CD: GitHub Actions
- Деплой: Docker + VPS (планируется)
